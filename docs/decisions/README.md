# Decyzje architektoniczne (ADR)

Tu twórz pliki `ADR-NNN.md` z decyzjami architektonicznymi projektu.

## Format

```markdown
# ADR-NNN: Tytuł decyzji

**Status:** ACTIVE | SUPERSEDED | DEPRECATED
**Data:** YYYY-MM-DD

## Kontekst
[Dlaczego ta decyzja jest potrzebna]

## Decyzja
[Co postanowiono]

## Konsekwencje
[Skutki tej decyzji]
```
