# Tasks and Tracking

Sprint tracking, backlog management, and progress reports. Agents use this folder to break down work, track what's done, and plan what's next.

## Starter Templates

- [`backlog.md`](backlog.md) — prioritized list of tasks (High / Medium / Low / Icebox)
- [`sprint-001.md`](sprint-001.md) — copy and rename to `sprint-NNN.md` for each sprint

## Suggested File Naming

- `backlog.md` — always the single source of backlog truth
- `sprint-NNN.md` — sprint planning and retrospective
- `done.md` — archive of completed items (optional)

## For Agents

When starting a task session, check `backlog.md` for priorities. Move items to the current sprint file when actively working on them. Mark items done with `[x]` and move them to `done.md` once complete. Update the retrospective section at the end of each sprint.
